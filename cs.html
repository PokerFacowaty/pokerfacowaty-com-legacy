Cheat Sheet

➤ shell

Ctrl + A Line beginning

Ctrl + E EOTL

Ctrl + L Clear the screen

Ctrl + U Clear the line before cursor

Ctrl + K Clear the line after cursor

Ctrl + R Search last commands

Alt + F Cursor one word forward

Alt + B Cursor one word backward

alias {short}="{full}" Create an alias

Save to .bashrc / .zshrc to make permanent

➤ irssi

Alt + N Switch window to N

Works until Alt + O

PgUp/PgDwn Scroll

/server add -ssl -auto -network {network} {link} {port} Auto connect

/network add -autosendcmd "/msg nickserv identify {password} ;wait 2000" {network} Auto identify

/channel add -auto {channel} {network} Auto join channels

/save Save all settings

➤ grep

grep {options} {string} {path} Searching for the string in the file

-i Ignore case

-r Dir and all subdirs

| OR operator

➤ ffmpeg

ffmpeg -ss {time} -i {input} -filter:v yadif=0:1:0 -c:v libx264 -crf 18 -c:a aac -b 320k {output} Hi8 for DaVinci

H.264 at 1920x1440 capped at 10 Mb/s

ffmpeg -ss {time} -i {input} -filter:v yadif=0:1:0 -c:v libx264 -crf 22 -s 1920x1440 -c:a aac -b 320k {output} Hi8 skipping DaVinci

-map {input number}:{v/a/s}:{stream number} Mapping
