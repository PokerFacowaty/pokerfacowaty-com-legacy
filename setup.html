➤ Music Sync
I ditched Spotify in favour of locally stored MP3s (320 kbps when possible) a long time ago and this is my current method of having all my music synced across devices
  • foobar2000 on my PC has the 'master' copy where I make changes to the music library and playlists
  • The library and the playlists are sent to the server using Syncthing
  • The server has a cronjob with a bash script (thanks, andrzej!) set up to regularly transform playlist files for use with Navidrome
  • Navidrome scans any changes to the library and the playlists and imports everything automatically
  • I use the Navidrome Web UI or iSub on other devices than my PC (since Navidrome follows the Subsonic standard, any Subsonic-compatible app will work)
